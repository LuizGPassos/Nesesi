Nesesi is an open-source web chat-bot project that utilizes the generative GPT-3.5 technology developed and provided by OpenAI to assist, guide, and teach new languages to anyone, anywhere, and on any device! The platform operates in a didactic, personalized, and adaptable manner, aiming to create an immersive and effective experience for the user, and is also accessible to people of all ages, with disabilities or disorders.

I will always be developing and evolving the platform, seeking to bring new features and incorporate the latest technologies. As the project is open-source, any and all help in development is welcome. If you are interested, look for the repository on my GitHub page.
